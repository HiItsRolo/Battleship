Battleship Rules

Before play begins, each player secretly arranges their ships on their primary grid. Each ship occupies a number of consecutive squares on the grid, arranged either horizontally or vertically. The number of squares for each ship is determined by the type of the ship. The ships cannot overlap so that only one ship can occupy any given square on the player's board. The types and numbers of ships allowed are the same for each player. They are one each of the following:

  1. Carrier (size=5)
  2. Battleship (size=4)
  3. Destroyer (size=3)
  4. Submarine (size=3)
  5. Patrol Boat (size=2)
Super Powers:
  • Send 3 torpedoes in one turn. (torpedos at different locations)
  • Send a big torpedo (a torpedo of size 5, with the shape of a + (middle, left, right, up, down)

Each player can use one of the 2 super powers once per game. They are enabled only if the player has 2 ships left on the board.
